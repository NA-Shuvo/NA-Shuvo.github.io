Problem 3: A Naive Geometer

Description


Shila is new in coordinate geometry. So she is very naive. She is facing problem in solving
a basic geometry problem. Shila has three integer coordinates. She needs to find the area of
the triangle enclosed by the connected edge of those coordinates. Help her to find the area.


Input

Input consists of three lines. Each line consists of two space separated integers.


Output

Output is only a floating point number with three decimal places after the radix point.


Sample Input 1

0 2
0 0
1 0


Sample Output 1

1.000


Sample Input 2

0 0
1 0
1 1


Sample Output 2

0.500
