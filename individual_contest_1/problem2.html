Problem 2: Missing Transitive Pairs

Description


Transitive relation is a special type of relation in Mathematics. A relation R on set X is
transitive if for all elements a,b,c in X, whenever R relates a to b and b to c, R must relate
a to c. As a non-mathematical example, the relation "is an ancestor of" is transitive. For
example, if Amy is an ancestor of Becky, and Becky is an ancestor of Carrie, then Amy, too, is
an ancestor of Carrie.

Now, for the purpose of this problem we denote the symbol T as the transitive relation operator
between two operands i.e. numbers. Now, let's say X = {1, 2, 3, 4, 5, 6, 7}. A relation R is
R is given bellow:

R = (1T2, 2T3, 4T7, 7T5, 7T6). According to the definition, R is NOT a transitive relation
because if 1T2 and 2T3, there must be 1T3 in R which is missing. Similarly the other 2 missing
pairs are 4T5 and 4T6.

In this problem, you will be given a relation R. You have to find out the missing pairs for R
to be transitive.


Input

The first line of input consists of an integer n where 0 < n <= 10000. The following n lines
will consist of a string as a pattern of relation such as '1T2', '533T612'.


Output

You have to print the missing pairs in xTy format, single pair in a line and must follow the
following rules:

1. Between two pairs xTy and wTz, if x > w, print pair xTy before pair wTz.
2. Between two pairs xTy and xTz, if y > z print pair xTy before pair xTz.
3. If there is NO missing pair, print No missing pair


Sample Input 1

5
1T2
2T3
4T7
7T5
7T6


Sample Output 1

1T3
4T6
4T5


Sample Input 2

4
1T2
2T3
1T3
5T6


Sample Output 2

No missing pair
