Problem 1: Integer Decomposition

Description

From number theory it is guaranted that any integer can be decomposed into the difference
of two square numbers, let's say x^2 and y^2 with (x>y). But this decomposition does not give
guarante x and y to be integers. For example, the number 13 can be decomposed as 7^2 - 6^2.
Here 7 and 6 are integers. But the number 26 doesn't have such integer decomposition. On the
other hand there are many integers having more than one integer square decomposiotions. 24 is
such a number. It can be decomposed as 7^2 - 5^2 and 5^2 - 1^2. So, it has two decompositions.

In this problem you will be given an integer and you have to tell the number of valid possible
integer square decomposiotions.


Input

Input consists of only one integer n such that 0 <= n <= 10^5.


Output

Only an integer that represents the number of valid possible integer square decomposiotions.
If infinite number of decompositions is possible, print inf.


Sample Input 1

5


Sample Output 1

1


Sample Input 2

24


Sample Output 2

2
